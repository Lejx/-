### 三、简答

**1.** 机器学习做图像处理的优势：(1) 自动化特征提取——传统方法依赖手工设计特征，需要专业知识；机器学习能从数据中自动学习。(2) 端到端学习——直接从输入到输出映射，简化流程。(3) 泛化能力——训练后能在未见数据上表现良好。(4) 跨领域通用——同一套方法可迁移到 NLP、语音等任务。

**2.** 区别：图像增强目的是改善视觉效果或突出感兴趣信息，不一定依据退化模型；图像复原根据退化模型（g=Hf+w）估计原始图像，更强调物理/数学建模。联系：两者都旨在提高图像质量；很多实际任务（如老照片修复）会同时用到增强与复原技术。

**3.** Marr/LoG 方法：(1) 选取高斯函数 G(x,y) 作为平滑滤波器；(2) 对图像 f(x,y) 进行高斯平滑卷积得到 g(x,y)；(3) 对平滑后的图像计算拉普拉斯（二阶导数），利用卷积与微分可交换的性质合并为 LoG 算子；(4) 找二阶导数的零交叉点作为边缘位置。LoG 算子同时完成平滑去噪和边缘检测，高斯因子 σ 越大，检测到的边缘越少、越粗。

**4.** 哈夫曼编码：概率大的符号分配短码字，概率小的分配长码字。步骤：①按概率从小到大排序；②合并两个最小概率节点为新节点；③重复直到构成二叉树；④从根到叶分配 0/1 得到码字。变长最佳编码定理：码字长度严格按符号概率大小逆序排列时，平均码长最小。

**5.** YOLO 核心思想：将目标检测视为回归问题，把图像划分为 S×S 网格，每个网格直接预测 B 个检测框的位置 (x,y,w,h)、置信度和 C 个类别概率。输出张量为 S²×(5B+C)。优点：速度快（实时检测 45fps）、背景误检率低。缺点：精度低于同期 Faster R-CNN、对小目标和密集目标效果差。

```python
import numpy as np

def laplacian_sharpen(image):
    """
    对灰度图像使用四邻域拉普拉斯算子进行锐化处理。
    image: 二维 NumPy 数组（灰度图像）
    返回: 锐化后的二维 NumPy 数组
    """
    # 复制原图，避免修改原始数据
    result = image.astype(np.float64).copy()
    rows, cols = image.shape

    # 对内部像素（不含边界）逐像素计算
    for i in range(1, rows - 1):
        for j in range(1, cols - 1):
            # 四邻域拉普拉斯算子
            laplacian = (image[i+1, j] + image[i-1, j]
                         + image[i, j+1] + image[i, j-1]
                         - 4 * image[i, j])
            # 锐化 = 原值 - 拉普拉斯值
            result[i, j] = image[i, j] - laplacian

    # 将结果裁剪到 [0, 255] 范围
    result = np.clip(result, 0, 255).astype(np.uint8)
    return result
```

### 三、简答

**1.** 可分离性：二维 DFT 可以拆分为先按行做一维 DFT，再按列做一维 DFT。意义：大幅降低计算复杂度，从 O(N⁴) 降低到 O(N²·N·logN)（配合 FFT），且可并行计算。

**2.** 稀疏连接：每个输出只与输入的局部区域相连，而非全连接，减少参数量。参数共享：同一个卷积核在整个特征图上滑动使用相同权重。等变表示：如果输入平移，输出也相应平移，保持空间对应关系。

**3.** NMS 用于去除目标检测中同一目标的冗余框。步骤：(1) 按置信度排序所有检测框；(2) 选出最高置信度的框保留；(3) 计算其余框与它的 IoU，删除 IoU>阈值的框；(4) 在剩余框中重复 (2)(3) 直到处理完毕。

**4.** 编码冗余：使用定长码表示不等概率符号造成的冗余，用变长编码（哈夫曼/算术编码）去除。像素间冗余：相邻像素高度相关，用预测编码、变换编码去除。心理视觉冗余：人眼不敏感的信息，通过量化去除（有损）。

**5.** MLE 只考虑似然函数 p(x|θ)，求使观测数据概率最大的 θ。MAP 同时考虑先验 p(θ) 和似然，求最大化 p(θ|x) ∝ p(x|θ)p(θ)。联系：当先验为均匀分布时 MAP 退化为 MLE。

```python
import numpy as np

def sobel_edge_detect(image):
    """
    Sobel 边缘检测。
    image: 二维 NumPy 数组（灰度图像）
    返回: 梯度幅值图像（二维数组）
    """
    image = image.astype(np.float64)
    rows, cols = image.shape

    # 定义 Sobel 算子
    sobel_x = np.array([[-1, 0, 1],
                        [-2, 0, 2],
                        [-1, 0, 1]], dtype=np.float64)

    sobel_y = np.array([[-1, -2, -1],
                        [ 0,  0,  0],
                        [ 1,  2,  1]], dtype=np.float64)

    # 初始化梯度图像
    grad_x = np.zeros_like(image)
    grad_y = np.zeros_like(image)

    # 对内部像素卷积
    for i in range(1, rows - 1):
        for j in range(1, cols - 1):
            region = image[i-1:i+2, j-1:j+2]
            grad_x[i, j] = np.sum(region * sobel_x)
            grad_y[i, j] = np.sum(region * sobel_y)

    # 计算梯度幅值
    magnitude = np.sqrt(grad_x**2 + grad_y**2)

    # 归一化到 [0, 255]
    magnitude = np.clip(magnitude, 0, 255).astype(np.uint8)
    return magnitude
```

### 三、简答

**1.** K-L 变换以数据协方差矩阵的特征向量为基进行正交变换，将数据投影到特征空间，使得变换后各分量不相关。变换后能量集中在前几个主分量上，因此被称为最优变换（能量集中能力最强）。在图像处理中用于降维（PCA）、人脸识别（Eigenfaces）、图像压缩等。

**2.** 逆滤波直接用 F̂=G/H 估计原图；但当 H 接近 0 时，噪声被放大，结果严重失真。维纳滤波引入噪声-信号功率谱比 (Sₙ/Sf)，在 H 较小处自动降低增益，从而在复原精度与噪声抑制间取得平衡。

**3.** 直方图均衡化的目的是使图像灰度分布更均匀，增强对比度。基本思想：利用累积分布函数（CDF）作为灰度变换函数 s=T(r)=(L-1)·CDF(r)，将集中的直方图拉伸到整个灰度范围。

**4.** Faster R-CNN 框架：(1) 骨干网络提取共享特征图；(2) RPN 在特征图上滑动产生候选区域（anchor-based）；(3) RoI Pooling 将候选区域映射到固定大小特征；(4) 全连接层进行分类和边界框回归。端到端可训练。

**5.** 贝叶斯最小错误率分类器：先验概率 P(ω) 是类别的先验分布；根据贝叶斯公式 P(ω|x)=p(x|ω)P(ω)/p(x) 计算后验概率；MAP 准则选择后验概率最大的类。当先验相等时退化为最大似然分类。

```python
import heapq

def huffman_encode(prob_dict):
    """
    哈夫曼编码。
    prob_dict: 字典，键为符号，值为概率，如 {'A':0.5, 'B':0.25, 'C':0.125, 'D':0.125}
    返回: 字典，键为符号，值为编码字符串
    """
    # 构建优先队列（最小堆），元素为 (概率, 唯一序号, 符号或子树)
    heap = []
    for i, (sym, prob) in enumerate(prob_dict.items()):
        heapq.heappush(heap, (prob, i, sym))

    counter = len(prob_dict)

    # 合并节点直到只剩一个根节点
    while len(heap) > 1:
        p1, _, left = heapq.heappop(heap)
        p2, _, right = heapq.heappop(heap)
        merged = (left, right)
        heapq.heappush(heap, (p1 + p2, counter, merged))
        counter += 1

    # 从根节点遍历二叉树，生成编码
    root = heap[0][2]
    codes = {}

    def traverse(node, code):
        if isinstance(node, str):
            # 叶子节点（符号）
            codes[node] = code if code else '0'  # 单符号特殊处理
            return
        traverse(node[0], code + '0')  # 左子树编码 0
        traverse(node[1], code + '1')  # 右子树编码 1

    traverse(root, '')
    return codes

# 示例
probs = {'A': 0.5, 'B': 0.25, 'C': 0.125, 'D': 0.125}
result = huffman_encode(probs)
for sym, code in sorted(result.items()):
    print(f"{sym}: {code}")
```

### 三、简答

**1.** Otsu 法：遍历所有可能阈值 t（0~L-1），将像素分为前景和背景两类。计算每个 t 下的类间方差 σ²_B = w₀w₁(μ₀-μ₁)²，选择使 σ²_B 最大的 t 作为最优阈值。最大化类间方差等价于最小化类内方差。

**2.** GLCM 定义了像素对 (i,j) 在指定方向和距离上的联合概率分布。四个常用特征：对比度——衡量灰度差异大小（值越大纹理越粗糙）；能量（角二阶矩）——衡量灰度均匀性；熵——衡量纹理复杂度/随机性；相关性——衡量灰度线性相关程度。

```python
import numpy as np

def conv2d(image, kernel):
    """
    通用二维卷积（valid 模式）。
    image: 二维 NumPy 数组（灰度图像）
    kernel: 二维 NumPy 数组（卷积核）
    返回: 卷积结果（valid 模式，不含边界）
    """
    image = image.astype(np.float64)
    kernel = kernel.astype(np.float64)

    # 获取图像和卷积核尺寸
    img_h, img_w = image.shape
    ker_h, ker_w = kernel.shape

    # 计算输出尺寸
    out_h = img_h - ker_h + 1
    out_w = img_w - ker_w + 1

    # 初始化输出
    output = np.zeros((out_h, out_w), dtype=np.float64)

    # 卷积运算（实际为相关运算）
    for i in range(out_h):
        for j in range(out_w):
            # 提取与卷积核等大的区域
            region = image[i:i+ker_h, j:j+ker_w]
            # 逐元素相乘再求和
            output[i, j] = np.sum(region * kernel)

    return output
```

### 三、简答

**1.** Canny 边缘检测步骤：(1) 高斯平滑去噪；(2) 用 Sobel 等算子计算梯度幅值和方向；(3) 非极大值抑制——在梯度方向上只保留局部最大值，细化边缘到单像素宽；(4) 双阈值检测——高阈值确定强边缘，低阈值确定弱边缘；(5) 滞后阈值跟踪——与强边缘相连的弱边缘保留，否则删除。

**2.** 迁移学习是将在大数据集（如 ImageNet）上预训练好的模型迁移到新任务上。常见做法：冻结预训练网络的卷积层作为特征提取器，只训练新的全连接层/分类头；或用较小学习率对整体微调（fine-tuning）。优势：减少训练数据需求，加速收敛，提高小数据集上的泛化能力。

**3.** 设原图灰度分布的 CDF 为 T(r)，目标为均匀分布。令 s=T(r)=(L-1)∫₀ʳ p(t)dt，由概率论知对连续分布 s 满足均匀分布。离散情况下 sₖ=round((L-1)·CDF(rₖ))，得到近似均匀分布。

**4.** Anchor 是预设的一组不同尺度和长宽比的参考框。在 Faster R-CNN 中，RPN 在每个特征图位置放置多个 anchor，网络学习预测每个 anchor 是否包含目标以及边界框的偏移量。YOLO 中每个网格也预设 anchor 模板。Anchor 的作用：将检测问题转化为对预定义框的分类和回归，简化学习任务。

**5.** SSIM 由三部分组成：亮度比较 l(x,y)、对比度比较 c(x,y)、结构比较 s(x,y)。SSIM = l·c·s，取值 [-1,1]，越接近 1 越相似。与 PSNR 区别：PSNR 基于逐像素差异（MSE），不考虑结构信息；SSIM 更符合人眼感知，考虑了局部结构、亮度和对比度。

### 五、综合

**1.** JPEG 流程：
(1) RGB→YCbCr，对 Cb/Cr 做 4:2:0 子采样（利用人眼对色度不敏感）；
(2) 将图像分为 8×8 块，每块做 DCT；
(3) 用量化表对 DCT 系数量化（有损步骤，人眼不敏感的高频系数被量化为 0）；
(4) 将量化后系数按 Zigzag 顺序排列，用 RLE 编码连续零值；
(5) 对 DC 系数做差分编码，对 AC 系数和 RLE 结果做哈夫曼/算术编码。
JPEG 为有损压缩，损失主要在量化步骤。

**2.**
(1) YOLO：单阶段，输入图像经骨干网络后，直接在输出特征图上预测框和类别，无需候选区域。Faster R-CNN：双阶段，第一阶段 RPN 生成候选区域，第二阶段对每个候选区域进行分类和回归。
(2) YOLO 速度快（可实时 45+ fps），但精度较低（尤其小目标）；Faster R-CNN 精度高，但速度慢（~5-7 fps）。原因：YOLO 一次前向传播完成，Faster R-CNN 需要两阶段处理。
(3) YOLO 适合实时检测（自动驾驶、视频监控）；Faster R-CNN 适合精度优先的场景（医学影像、安防）。

```python
import numpy as np

def histogram_equalization(image):
    """
    灰度图像直方图均衡化。
    image: 二维 NumPy 数组，灰度值 0-255
    返回: 均衡化后的图像
    """
    # 1. 统计直方图
    hist = np.zeros(256, dtype=np.int64)
    rows, cols = image.shape
    for i in range(rows):
        for j in range(cols):
            hist[image[i, j]] += 1

    # 2. 计算概率分布
    total_pixels = rows * cols
    pdf = hist / total_pixels

    # 3. 计算累积分布函数 CDF
    cdf = np.cumsum(pdf)

    # 4. 建立灰度映射表
    mapping = np.round(255 * cdf).astype(np.uint8)

    # 5. 应用映射
    result = mapping[image]
    return result
```
