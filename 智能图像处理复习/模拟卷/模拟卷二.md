### 一、单选

> 第 5 题解析：2D-DFT 可分离为先按行做 N 次一维 DFT（共 M 行），再按列做 M 次一维 DFT（共 N 列），总共 M+N 次。

### 二、多选

1-ABD　2-ABD　3-ABCD　4-ABC　5-ABC　6-ABD　7-ABC　8-ABC　9-ABC　10-ABC

> 多选第 2 题：沃尔什变换能量集中能力弱于 K-L 变换，C 错误。
> 多选第 5 题：mAP 值与 IoU 阈值有关（如 mAP@0.5 与 mAP@0.75 不同），D 错误。
> 多选第 10 题：减少训练数据会导致过拟合加重，D 错误。

### 三、简答

**1.** K-L 变换以数据协方差矩阵的特征向量为基进行正交变换，将数据投影到特征空间，使得变换后各分量不相关。变换后能量集中在前几个主分量上，因此被称为最优变换（能量集中能力最强）。在图像处理中用于降维（PCA）、人脸识别（Eigenfaces）、图像压缩等。

**2.** 逆滤波直接用 F̂=G/H 估计原图；但当 H 接近 0 时，噪声被放大，结果严重失真。维纳滤波引入噪声-信号功率谱比 (Sₙ/Sf)，在 H 较小处自动降低增益，从而在复原精度与噪声抑制间取得平衡。

**3.** 直方图均衡化的目的是使图像灰度分布更均匀，增强对比度。基本思想：利用累积分布函数（CDF）作为灰度变换函数 s=T(r)=(L-1)·CDF(r)，将集中的直方图拉伸到整个灰度范围。

**4.** Faster R-CNN 框架：(1) 骨干网络提取共享特征图；(2) RPN 在特征图上滑动产生候选区域（anchor-based）；(3) RoI Pooling 将候选区域映射到固定大小特征；(4) 全连接层进行分类和边界框回归。端到端可训练。

**5.** 贝叶斯最小错误率分类器：先验概率 P(ω) 是类别的先验分布；根据贝叶斯公式 P(ω|x)=p(x|ω)P(ω)/p(x) 计算后验概率；MAP 准则选择后验概率最大的类。当先验相等时退化为最大似然分类。

### 四、计算

**1.**
(1) 量化：round(8/4)=2, round(-2/2)=-1, round(1/2)=1（四舍五入0.5取1）, round(0/1)=0 → 量化矩阵 [[2,-1],[1,0]]
（注：round(0.5) 在不同规则下可能为 0 或 1，此处按四舍五入取 1）
(2) 反量化：2×4=8, -1×2=-2, 1×2=2, 0×1=0 → [[8,-2],[2,0]]
(3) 量化去除的是心理视觉冗余（人眼不敏感的高频信息），是有损过程，引入量化误差（此例中 (1,0) 位置从 1 变为 2）。

**2.**
(1) 距离 = |w₁x₁+w₂x₂+b|/√(w₁²+w₂²)：(1,1): |2|/√2=√2; (2,2): |4|/√2=2√2; (-1,-1): |-2|/√2=√2; (0,-1): |-1|/√2=√2/2
(2) 支持向量：距离超平面最近的点。正类中 (1,1)，负类中 (0,-1)（距离分别为 √2 和 √2/2），实际支持向量应为两侧距离最近的各自代表，即 (1,1) 和 (0,-1)。
(3) M = 2×min(正类最近距离, 负类最近距离)… 更准确地说，分类间隔 = 2/||w|| = 2/√2 = √2（如果假定超平面已经归一化为标准形式 |w·x+b|=1 对支持向量）。

**3.**
(1) F̂(u,v) = G(u,v)/H(u,v)
(2) 当 H(u,v)→0 时，G/H → ∞，噪声 N/H 被无限放大，复原图像被噪声淹没。
(3) 维纳滤波：F̂(u,v) = [H*(u,v) / (|H|² + Sₙ/Sf)] · G(u,v)。当 |H| 较大时近似逆滤波；当 |H| 较小时分母中 Sₙ/Sf 占主导，降低增益，避免噪声放大。

**4.**
(1) P(x) = 0.3×0.6 + 0.8×0.4 = 0.18 + 0.32 = 0.50
(2) P(ω₁|x) = 0.18/0.50 = 0.36; P(ω₂|x) = 0.32/0.50 = 0.64
(3) P(ω₂|x) > P(ω₁|x)，根据 MAP 准则，x 归为 ω₂ 类。

```python
import heapq

def huffman_encode(prob_dict):
    """
    哈夫曼编码。
    prob_dict: 字典，键为符号，值为概率，如 {'A':0.5, 'B':0.25, 'C':0.125, 'D':0.125}
    返回: 字典，键为符号，值为编码字符串
    """
    # 构建优先队列（最小堆），元素为 (概率, 唯一序号, 符号或子树)
    heap = []
    for i, (sym, prob) in enumerate(prob_dict.items()):
        heapq.heappush(heap, (prob, i, sym))

    counter = len(prob_dict)

    # 合并节点直到只剩一个根节点
    while len(heap) > 1:
        p1, _, left = heapq.heappop(heap)
        p2, _, right = heapq.heappop(heap)
        merged = (left, right)
        heapq.heappush(heap, (p1 + p2, counter, merged))
        counter += 1

    # 从根节点遍历二叉树，生成编码
    root = heap[0][2]
    codes = {}

    def traverse(node, code):
        if isinstance(node, str):
            # 叶子节点（符号）
            codes[node] = code if code else '0'  # 单符号特殊处理
            return
        traverse(node[0], code + '0')  # 左子树编码 0
        traverse(node[1], code + '1')  # 右子树编码 1

    traverse(root, '')
    return codes

# 示例
probs = {'A': 0.5, 'B': 0.25, 'C': 0.125, 'D': 0.125}
result = huffman_encode(probs)
for sym, code in sorted(result.items()):
    print(f"{sym}: {code}")
```
