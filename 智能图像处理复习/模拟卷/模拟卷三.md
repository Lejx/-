### 三、简答

**1.** 可分离性：二维 DFT 可以拆分为先按行做一维 DFT，再按列做一维 DFT。意义：大幅降低计算复杂度，从 O(N⁴) 降低到 O(N²·N·logN)（配合 FFT），且可并行计算。

**2.** 稀疏连接：每个输出只与输入的局部区域相连，而非全连接，减少参数量。参数共享：同一个卷积核在整个特征图上滑动使用相同权重。等变表示：如果输入平移，输出也相应平移，保持空间对应关系。

**3.** NMS 用于去除目标检测中同一目标的冗余框。步骤：(1) 按置信度排序所有检测框；(2) 选出最高置信度的框保留；(3) 计算其余框与它的 IoU，删除 IoU>阈值的框；(4) 在剩余框中重复 (2)(3) 直到处理完毕。

**4.** 编码冗余：使用定长码表示不等概率符号造成的冗余，用变长编码（哈夫曼/算术编码）去除。像素间冗余：相邻像素高度相关，用预测编码、变换编码去除。心理视觉冗余：人眼不敏感的信息，通过量化去除（有损）。

**5.** MLE 只考虑似然函数 p(x|θ)，求使观测数据概率最大的 θ。MAP 同时考虑先验 p(θ) 和似然，求最大化 p(θ|x) ∝ p(x|θ)p(θ)。联系：当先验为均匀分布时 MAP 退化为 MLE。

### 四、计算

(3) 乘以 (-1)^(x+y) 等价于将频谱中心化（将零频移到中心），即平移 (M/2, N/2)。

(2) C₂ = 40²/100 = 16; 16 < 25 且更接近 12.57，故第二个目标更接近圆形。

```python
import numpy as np

def sobel_edge_detect(image):
    """
    Sobel 边缘检测。
    image: 二维 NumPy 数组（灰度图像）
    返回: 梯度幅值图像（二维数组）
    """
    image = image.astype(np.float64)
    rows, cols = image.shape

    # 定义 Sobel 算子
    sobel_x = np.array([[-1, 0, 1],
                        [-2, 0, 2],
                        [-1, 0, 1]], dtype=np.float64)

    sobel_y = np.array([[-1, -2, -1],
                        [ 0,  0,  0],
                        [ 1,  2,  1]], dtype=np.float64)

    # 初始化梯度图像
    grad_x = np.zeros_like(image)
    grad_y = np.zeros_like(image)

    # 对内部像素卷积
    for i in range(1, rows - 1):
        for j in range(1, cols - 1):
            region = image[i-1:i+2, j-1:j+2]
            grad_x[i, j] = np.sum(region * sobel_x)
            grad_y[i, j] = np.sum(region * sobel_y)

    # 计算梯度幅值
    magnitude = np.sqrt(grad_x**2 + grad_y**2)

    # 归一化到 [0, 255]
    magnitude = np.clip(magnitude, 0, 255).astype(np.uint8)
    return magnitude
```
