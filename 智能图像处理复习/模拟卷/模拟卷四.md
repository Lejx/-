### 三、简答

**1.** Canny 边缘检测步骤：(1) 高斯平滑去噪；(2) 用 Sobel 等算子计算梯度幅值和方向；(3) 非极大值抑制——在梯度方向上只保留局部最大值，细化边缘到单像素宽；(4) 双阈值检测——高阈值确定强边缘，低阈值确定弱边缘；(5) 滞后阈值跟踪——与强边缘相连的弱边缘保留，否则删除。

**2.** 迁移学习是将在大数据集（如 ImageNet）上预训练好的模型迁移到新任务上。常见做法：冻结预训练网络的卷积层作为特征提取器，只训练新的全连接层/分类头；或用较小学习率对整体微调（fine-tuning）。优势：减少训练数据需求，加速收敛，提高小数据集上的泛化能力。

**3.** 设原图灰度分布的 CDF 为 T(r)，目标为均匀分布。令 s=T(r)=(L-1)∫₀ʳ p(t)dt，由概率论知对连续分布 s 满足均匀分布。离散情况下 sₖ=round((L-1)·CDF(rₖ))，得到近似均匀分布。

**4.** Anchor 是预设的一组不同尺度和长宽比的参考框。在 Faster R-CNN 中，RPN 在每个特征图位置放置多个 anchor，网络学习预测每个 anchor 是否包含目标以及边界框的偏移量。YOLO 中每个网格也预设 anchor 模板。Anchor 的作用：将检测问题转化为对预定义框的分类和回归，简化学习任务。

**5.** SSIM 由三部分组成：亮度比较 l(x,y)、对比度比较 c(x,y)、结构比较 s(x,y)。SSIM = l·c·s，取值 [-1,1]，越接近 1 越相似。与 PSNR 区别：PSNR 基于逐像素差异（MSE），不考虑结构信息；SSIM 更符合人眼感知，考虑了局部结构、亮度和对比度。

### 五、综合

**1.** JPEG 流程：
(1) RGB→YCbCr，对 Cb/Cr 做 4:2:0 子采样（利用人眼对色度不敏感）；
(2) 将图像分为 8×8 块，每块做 DCT；
(3) 用量化表对 DCT 系数量化（有损步骤，人眼不敏感的高频系数被量化为 0）；
(4) 将量化后系数按 Zigzag 顺序排列，用 RLE 编码连续零值；
(5) 对 DC 系数做差分编码，对 AC 系数和 RLE 结果做哈夫曼/算术编码。
JPEG 为有损压缩，损失主要在量化步骤。

**2.**
(1) YOLO：单阶段，输入图像经骨干网络后，直接在输出特征图上预测框和类别，无需候选区域。Faster R-CNN：双阶段，第一阶段 RPN 生成候选区域，第二阶段对每个候选区域进行分类和回归。
(2) YOLO 速度快（可实时 45+ fps），但精度较低（尤其小目标）；Faster R-CNN 精度高，但速度慢（~5-7 fps）。原因：YOLO 一次前向传播完成，Faster R-CNN 需要两阶段处理。
(3) YOLO 适合实时检测（自动驾驶、视频监控）；Faster R-CNN 适合精度优先的场景（医学影像、安防）。

```python
import numpy as np

def histogram_equalization(image):
    """
    灰度图像直方图均衡化。
    image: 二维 NumPy 数组，灰度值 0-255
    返回: 均衡化后的图像
    """
    # 1. 统计直方图
    hist = np.zeros(256, dtype=np.int64)
    rows, cols = image.shape
    for i in range(rows):
        for j in range(cols):
            hist[image[i, j]] += 1

    # 2. 计算概率分布
    total_pixels = rows * cols
    pdf = hist / total_pixels

    # 3. 计算累积分布函数 CDF
    cdf = np.cumsum(pdf)

    # 4. 建立灰度映射表
    mapping = np.round(255 * cdf).astype(np.uint8)

    # 5. 应用映射
    result = mapping[image]
    return result
```
