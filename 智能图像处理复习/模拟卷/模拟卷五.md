### 三、简答

**1.** Otsu 法：遍历所有可能阈值 t（0~L-1），将像素分为前景和背景两类。计算每个 t 下的类间方差 σ²_B = w₀w₁(μ₀-μ₁)²，选择使 σ²_B 最大的 t 作为最优阈值。最大化类间方差等价于最小化类内方差。

**2.** GLCM 定义了像素对 (i,j) 在指定方向和距离上的联合概率分布。四个常用特征：对比度——衡量灰度差异大小（值越大纹理越粗糙）；能量（角二阶矩）——衡量灰度均匀性；熵——衡量纹理复杂度/随机性；相关性——衡量灰度线性相关程度。

**3.** MLE 求使 p(X|θ) 最大的 θ；MAP 求使 p(θ|X)∝p(X|θ)p(θ) 最大的 θ。MAP 额外利用先验 p(θ)。当训练数据少时，先验能起正则化作用，MAP 优于 MLE。例如：抛硬币 3 次全正面，MLE 估计 θ=1，MAP 若用 Beta 先验则估计 θ<1 更合理。

**4.** 感受野指 CNN 中某层特征图上一个神经元所能"看到"的输入图像区域大小。增大方法：增加卷积层数（堆叠多层小卷积核）、使用更大的卷积核、使用带步长的卷积或池化、使用空洞卷积（dilated convolution）。

**5.** Mask R-CNN 在 Faster R-CNN 的分类 + 回归两个分支基础上，增加了第三个分支——掩码预测分支，对每个 RoI 用 FCN 输出逐像素的二值掩码。同时用 RoI Align 替代 RoI Pooling 消除量化误差，提高掩码精度。三个任务并行预测，实现端到端实例分割。

**1.**
数据：(2,3), (3,5), (4,4), (5,7), (6,6)
(1) 均值 x̄ = (2+3+4+5+6)/5=4, ȳ = (3+5+4+7+6)/5=5 → μ=[4,5]ᵀ
(2) 协方差矩阵（N-1 除法）：
Var(x) = [(2-4)²+(3-4)²+(4-4)²+(5-4)²+(6-4)²]/4 = (4+1+0+1+4)/4 = 2.5
Var(y) = [(3-5)²+(5-5)²+(4-5)²+(7-5)²+(6-5)²]/4 = (4+0+1+4+1)/4 = 2.5
Cov(x,y) = [(2-4)(3-5)+(3-4)(5-5)+(4-4)(4-5)+(5-4)(7-5)+(6-4)(6-5)]/4 = [4+0+0+2+2]/4 = 2
C = [[2.5, 2],[2, 2.5]]
(3) 特征方程 |C-λI|=0: (2.5-λ)²-4=0 → λ²-5λ+2.25=0 → λ=(5±√(25-9))/2=(5±4)/2
λ₁=4.5, λ₂=0.5

```python
import numpy as np

def conv2d(image, kernel):
    """
    通用二维卷积（valid 模式）。
    image: 二维 NumPy 数组（灰度图像）
    kernel: 二维 NumPy 数组（卷积核）
    返回: 卷积结果（valid 模式，不含边界）
    """
    image = image.astype(np.float64)
    kernel = kernel.astype(np.float64)

    # 获取图像和卷积核尺寸
    img_h, img_w = image.shape
    ker_h, ker_w = kernel.shape

    # 计算输出尺寸
    out_h = img_h - ker_h + 1
    out_w = img_w - ker_w + 1

    # 初始化输出
    output = np.zeros((out_h, out_w), dtype=np.float64)

    # 卷积运算（实际为相关运算）
    for i in range(out_h):
        for j in range(out_w):
            # 提取与卷积核等大的区域
            region = image[i:i+ker_h, j:j+ker_w]
            # 逐元素相乘再求和
            output[i, j] = np.sum(region * kernel)

    return output
```
