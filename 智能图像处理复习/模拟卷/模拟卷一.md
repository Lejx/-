### 三、简答

**1.** 机器学习做图像处理的优势：(1) 自动化特征提取——传统方法依赖手工设计特征，需要专业知识；机器学习能从数据中自动学习。(2) 端到端学习——直接从输入到输出映射，简化流程。(3) 泛化能力——训练后能在未见数据上表现良好。(4) 跨领域通用——同一套方法可迁移到 NLP、语音等任务。

**2.** 区别：图像增强目的是改善视觉效果或突出感兴趣信息，不一定依据退化模型；图像复原根据退化模型（g=Hf+w）估计原始图像，更强调物理/数学建模。联系：两者都旨在提高图像质量；很多实际任务（如老照片修复）会同时用到增强与复原技术。

**3.** Marr/LoG 方法：(1) 选取高斯函数 G(x,y) 作为平滑滤波器；(2) 对图像 f(x,y) 进行高斯平滑卷积得到 g(x,y)；(3) 对平滑后的图像计算拉普拉斯（二阶导数），利用卷积与微分可交换的性质合并为 LoG 算子；(4) 找二阶导数的零交叉点作为边缘位置。LoG 算子同时完成平滑去噪和边缘检测，高斯因子 σ 越大，检测到的边缘越少、越粗。

**4.** 哈夫曼编码：概率大的符号分配短码字，概率小的分配长码字。步骤：①按概率从小到大排序；②合并两个最小概率节点为新节点；③重复直到构成二叉树；④从根到叶分配 0/1 得到码字。变长最佳编码定理：码字长度严格按符号概率大小逆序排列时，平均码长最小。

**5.** YOLO 核心思想：将目标检测视为回归问题，把图像划分为 S×S 网格，每个网格直接预测 B 个检测框的位置 (x,y,w,h)、置信度和 C 个类别概率。输出张量为 S²×(5B+C)。优点：速度快（实时检测 45fps）、背景误检率低。缺点：精度低于同期 Faster R-CNN、对小目标和密集目标效果差。

```python
import numpy as np

def laplacian_sharpen(image):
    """
    对灰度图像使用四邻域拉普拉斯算子进行锐化处理。
    image: 二维 NumPy 数组（灰度图像）
    返回: 锐化后的二维 NumPy 数组
    """
    # 复制原图，避免修改原始数据
    result = image.astype(np.float64).copy()
    rows, cols = image.shape

    # 对内部像素（不含边界）逐像素计算
    for i in range(1, rows - 1):
        for j in range(1, cols - 1):
            # 四邻域拉普拉斯算子
            laplacian = (image[i+1, j] + image[i-1, j]
                         + image[i, j+1] + image[i, j-1]
                         - 4 * image[i, j])
            # 锐化 = 原值 - 拉普拉斯值
            result[i, j] = image[i, j] - laplacian

    # 将结果裁剪到 [0, 255] 范围
    result = np.clip(result, 0, 255).astype(np.uint8)
    return result
```
