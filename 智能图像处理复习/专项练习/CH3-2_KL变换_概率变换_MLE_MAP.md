### 三、简答

**1.** K-L 变换步骤：(1) 计算数据的均值向量和协方差矩阵 C；(2) 对 C 做特征分解得到特征值和特征向量；(3) 按特征值从大到小排列，取前 q 个特征向量构成变换矩阵 A；(4) 投影 y = A(x-μ)。"最优"是指在均方误差意义下，用 q 个分量近似原数据时误差最小（即能量集中能力最强）。困难：需要对每组数据单独计算协方差矩阵和特征分解，计算量大，不像 DCT 有固定的变换核。

**2.** MLE 步骤：(1) 建立参数化概率模型 p(x|θ)；(2) 写出似然函数 L(θ)=∏p(xᵢ|θ)；(3) 取对数得 lnL(θ)=Σlnp(xᵢ|θ)；(4) 对 θ 求导令其为 0 求解。正态分布：μ_MLE = (1/n)Σxᵢ = x̄，σ²_MLE = (1/n)Σ(xᵢ-x̄)²。

**3.** 先验概率是在观测数据之前对参数的初始信念（如患病率 0.1‰），后验概率是结合观测数据后对参数的更新信念。新冠例子：P(患|阳) = P(阳|患)P(患)/P(阳) = 99.99%×0.01% / (99.99%×0.01%+0.01%×99.99%) ≈ 50%。因为患病率极低（先验概率极小），即使试纸准确率很高，阳性检测结果中真阳性的比例也不高——先验概率对后验概率影响巨大。

### 四、计算

**1.**
(1) μ = [(1+2+3+4+5)/5, (2+4+3+5+6)/5] = [3, 4]
(2) C = (1/4)Σ(xᵢ-μ)(xᵢ-μ)ᵀ
分量计算：Var(x₁) = [(1-3)²+(2-3)²+0+(1)²+(2)²]/4 = (4+1+0+1+4)/4 = 2.5
Var(x₂) = [(2-4)²+(4-4)²+(3-4)²+(5-4)²+(6-4)²]/4 = (4+0+1+1+4)/4 = 2.5
Cov(x₁,x₂) = [(1-3)(2-4)+(2-3)(4-4)+(3-3)(3-4)+(4-3)(5-4)+(5-3)(6-4)]/4 = [4+0+0+1+4]/4 = 2.25
C = [[2.5, 2.25],[2.25, 2.5]]
(3) |C-λI|=0: (2.5-λ)²-2.25²=0 → λ²-5λ+6.25-5.0625=0 → λ²-5λ+1.1875=0
λ = (5±√(25-4.75))/2 = (5±√20.25)/2 = (5±4.5)/2 → λ₁=4.75, λ₂=0.25
λ₁=4.75 对应特征向量：v₁∝[1,1]ᵀ（归一化为 [1/√2, 1/√2]ᵀ）
λ₂=0.25 对应特征向量：v₂∝[1,-1]ᵀ
(4) 保留方差比例 = λ₁/(λ₁+λ₂) = 4.75/5 = 95%

**2.**
(1) L(θ) = C(5,4)·θ⁴·(1-θ)¹ = 5θ⁴(1-θ)
(2) lnL = ln5+4lnθ+ln(1-θ)，求导：4/θ - 1/(1-θ) = 0 → 4(1-θ)=θ → θ_MLE = 4/5 = 0.8
(3) 后验 ∝ L(θ)·p(θ) = 5θ⁴(1-θ)·θ(1-θ) = 5θ⁵(1-θ)²
lnP ∝ 5lnθ+2ln(1-θ)，求导：5/θ-2/(1-θ)=0 → 5(1-θ)=2θ → θ_MAP = 5/7 ≈ 0.714
(4) MLE 给出 0.8，MAP 给出 0.714。先验 Beta(2,2) 偏向 θ=0.5（对称先验），使得 MAP 估计相比 MLE 向 0.5 "收缩"，起到正则化作用。当数据量增大时，MAP 和 MLE 趋于一致。
