### 三、简答

**1.** 软间隔 SVM 允许部分样本违反间隔约束。引入松弛变量 ξᵢ≥0 表示第 i 个样本的违反程度，目标函数变为 min (1/2)||w||² + CΣξᵢ。C 越大，对错分样本惩罚越重，间隔越小，模型越复杂（可能过拟合）；C 越小，允许更多错分，间隔更大，模型更简单（可能欠拟合）。

**2.** CNN 结构：卷积层使用多个滤波器与输入做卷积提取局部特征（边缘、纹理等）；池化层对特征图下采样（如 2×2 最大池化），减小尺寸和参数量，增强平移不变性；全连接层将特征展平后进行分类/回归。参数共享指同一滤波器在整个特征图上滑动使用相同权重，大幅减少参数量。稀疏连接指输出只与输入的局部区域相连（感受野），而非全连接。

**3.** 反向传播是训练神经网络的核心算法。前向传播：输入数据从输入层到隐藏层再到输出层，逐层计算激活值，最终得到预测值和损失。反向传播：从输出层开始，利用链式法则逐层计算损失对每个权重的梯度，然后用梯度下降更新权重。两层网络示例：输入 x → 隐层 h=σ(W₁x+b₁) → 输出 ŷ=W₂h+b₂，前向计算 ŷ 和 Loss；反向传播 ∂L/∂W₂=∂L/∂ŷ·h，∂L/∂W₁=∂L/∂h·σ'·x，从而更新 W₁、W₂。

### 四、计算

**1.**
n=4, Σx=10, Σy=17, Σxy=1×2+2×3+3×5+4×7=51, Σx²=1+4+9+16=30
a = (4×51-10×17)/(4×30-10²) = (204-170)/(120-100) = 34/20 = 1.7
x̄=2.5, ȳ=4.25, b = 4.25-1.7×2.5 = 4.25-4.25 = 0
回归方程：y = 1.7x

**2.**
(1) 每个卷积核参数 = 5×5×3 = 75，加偏置 = 76。16 个核共 16×76 = **1216** 个参数。
(2) 输出尺寸 = (32-5+2×2)/1 + 1 = 32。输出特征图为 32×32×16。
(3) 2×2 最大池化（stride=2）后：32/2 = 16。池化输出为 **16×16×16**。
