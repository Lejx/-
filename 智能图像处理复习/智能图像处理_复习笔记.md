# 智能图像处理复习笔记

## 复习主线

本课程可以按“图像基础 -> 视觉与采集 -> 图像变换 -> 机器学习/CNN -> 压缩编码 -> 增强复原 -> 分割 -> 描述分类 -> 检测与实例分割”来串联。

- 图像基础解决“图像是什么、数字图像如何表示、处理任务有哪些”。
- 图像变换解决“换到另一个域后更容易分析、压缩或去噪”。
- 机器学习与神经网络解决“从数据中自动学习特征和决策函数”。
- 压缩编码解决“如何用更少比特表示图像，同时控制失真”。
- 增强复原解决“如何让图像更清晰、更接近原始场景”。
- 分割、描述、分类解决“如何从图像中提取对象并识别其类别”。
- 目标检测与实例分割解决“对象在哪里、是什么、每个实例的像素范围是什么”。

## 第 1 章 绪论

### 1. 图像与数字图像

图像是人对视觉感知的物质再现。数字图像通常表示为二维离散函数 `f(x, y)`，其中 `(x, y)` 是像素坐标，`f` 是灰度或颜色值。彩色图像可看作多通道图像，如 RGB 三通道。

关键概念：

- 像素：数字图像的最小采样单元。
- 灰度级：像素亮度的离散取值，常见 8 bit 图像为 0-255。
- 空间分辨率：单位长度或整幅图像中可区分的像素数量。
- 灰度分辨率：灰度级数，通常由量化位数决定。
- 数字图像处理：用计算机对图像进行增强、复原、压缩、分割、识别等操作。

### 2. 图像处理目标

常见目标包括：

- 改善视觉效果：增强对比度、锐化边缘、伪彩色显示。
- 便于存储传输：图像压缩编码。
- 提取信息：分割、描述、分类、检测。
- 恢复退化图像：去噪、去模糊、几何校正。
- 支撑智能决策：机器学习、深度学习、目标检测与实例分割。

### 3. 空间域与变换域

空间域直接处理像素；变换域先将图像变换为频率或其他系数表示，再处理系数。傅里叶变换、沃尔什/哈达玛变换、DCT、K-L 变换都属于变换域方法。变换域常用于压缩、滤波、特征分析。

## 第 2 章 图像信息基本知识

### 1. 图像获取与显示

图像信息系统通常包含采集、传输、处理、显示等环节。图像采集会受到传感器、光照、采样、量化、噪声等因素影响；显示则涉及亮度、颜色、分辨率和人眼视觉特性。

### 2. 人眼视觉与图像技术

图像处理常利用人眼视觉特性：

- 人眼对亮度变化、边缘和对比度敏感。
- 人眼对不同空间频率的敏感度不同。
- 人眼对颜色的感知可用三基色原理解释。
- 视觉系统存在亮度适应、对比度敏感、掩蔽效应等现象。

### 3. 三基色与光度学

三基色原理认为多数颜色可由红、绿、蓝三种基色混合得到。图像显示与颜色处理常涉及 RGB、亮度、色度等概念。亮度是人眼感知强弱，光强是物理量，二者不完全等价。

## 第 3 章 图像变换

### 1. 图像线性运算

图像线性运算包括加权求和、线性滤波、卷积等。其核心是利用线性系统分析方法描述图像变换：

- 加法可用于图像叠加、降噪平均。
- 乘法或加权可用于亮度调整、模板操作。
- 卷积可表示空间滤波。

### 2. 二维离散傅里叶变换 DFT

傅里叶变换把图像从空间域转换到频率域。低频对应灰度缓慢变化和主体轮廓，高频对应边缘、纹理、噪声等快速变化。

重要性质：

- 可分离性：二维 DFT 可分解为行、列两个一维 DFT。
- 平移性质：空间平移引起频域相位变化。
- 卷积定理：空间域卷积对应频域乘法。
- 能量集中：自然图像大量能量常集中在低频区域。

复习重点：

- 低通滤波用于平滑去噪，但会模糊边缘。
- 高通滤波用于锐化边缘，但会增强噪声。
- 频谱中心化常通过乘以 `(-1)^(x+y)` 实现。

### 3. 沃尔什与哈达玛变换

沃尔什变换使用只取 `+1` 和 `-1` 的正交基函数，计算简单。哈达玛变换可看作特殊排序的沃尔什变换，具有递推结构，核矩阵元素也只有 `+1/-1`。

特点：

- 变换核可分离、对称。
- 不需要乘法或三角函数，适合快速硬件实现。
- 基函数呈块状变化，不如 DCT/Fourier 平滑。

### 4. 离散余弦变换 DCT

DCT 使用实数余弦基函数，是 JPEG 压缩中的核心变换。DCT 具有较强能量集中能力，很多自然图像经过 DCT 后能量集中在低频系数。

复习重点：

- DCT 是 K-L 变换的较好近似。
- DCT 系数左上角常为低频，右下角常为高频。
- JPEG 常按块进行 DCT、量化、熵编码。

### 5. K-L 变换与 PCA

K-L 变换又称 Hotelling 变换，与 PCA 密切相关。它基于数据协方差矩阵的特征向量构造正交基，实现去相关和能量集中。

特点：

- 理论上具有最佳去相关和均方误差意义下的能量压缩能力。
- 变换基依赖数据统计特性。
- 实用困难是需要估计协方差矩阵，计算代价较高。

### 6. 图像概率变换与参数估计

真实图像分布高维、复杂、非线性，常先假设简单基础分布，再通过变换描述真实分布。

关键概念：

- 高斯分布：由均值和方差/协方差刻画，是常用基础模型。
- 线性变换：高斯分布经过线性变换后仍为高斯分布。
- 非线性可逆变换：需要用雅可比行列式修正概率密度。
- MLE：最大化观测样本出现的似然。
- MAP：结合似然和先验，最大化后验概率。

## 第 4 章 机器学习与神经网络

### 1. 为什么用机器学习做图像处理

传统方法依赖人工设计特征，如边缘、纹理、形状、变换系数。机器学习能从数据中自动学习特征与决策规则，适合复杂场景。

优势：

- 自动化特征提取。
- 端到端学习。
- 对新数据具有泛化能力。
- 可与大规模标注数据和 GPU 计算结合。

### 2. 机器学习基本要素

Mitchell 定义：若程序在任务 `T` 上的性能度量 `P` 随经验 `E` 改善，则称它从经验中学习。

常用术语：

- 样本：被处理对象的量化表示。
- 特征：样本的属性向量。
- 标签：监督学习中的目标输出。
- 模型：从输入到输出的函数。
- 损失函数：衡量预测和真实值差异。
- 优化算法：更新参数以降低损失。

### 3. 线性回归

线性回归用 `y_hat = w^T x + b` 预测连续值。训练目标通常是最小化平方损失。

参数学习方法：

- 经验风险最小化：最小化训练集平均损失。
- 最小二乘法：平方损失下的闭式求解。
- 梯度下降/LMS：按负梯度方向迭代更新参数。
- 结构风险最小化：在经验风险上加入正则化项。
- MLE/MAP/贝叶斯估计：从概率角度解释参数估计。

### 4. 支持向量机 SVM

SVM 的目标是在特征空间中找到最大间隔分离超平面。支持向量是决定分界面的关键样本。

重点：

- 硬间隔适合线性可分数据。
- 软间隔引入松弛变量，允许部分样本违反间隔。
- 核函数把内积替换为高维特征空间中的相似度，可处理非线性分类。
- 常见核函数包括线性核、多项式核、RBF 核。

### 5. 神经元与激活函数

人工神经元通常计算加权和加偏置，再经过非线性激活函数输出。非线性激活使多层网络能表达非线性映射。

常见激活函数：

- Sigmoid/Logistic：输出在 0-1，可解释为概率，但易梯度饱和。
- Tanh：输出在 -1 到 1，零中心。
- Softmax：多分类输出归一化为概率分布。
- ReLU：`max(0, x)`，计算简单，缓解梯度消失。
- Leaky ReLU/PReLU/ELU/Softplus：改进 ReLU 的死神经元或平滑性问题。

### 6. 前馈网络与反向传播

前馈网络按层传播：

`z^(l) = W^(l) a^(l-1) + b^(l)`，`a^(l) = f(z^(l))`。

反向传播基于链式法则计算损失函数对各层参数的梯度，再用梯度下降或其变体更新参数。它是训练多层神经网络的核心算法。

### 7. 卷积神经网络 CNN

CNN 适合图像处理，因为卷积引入了图像局部性和平移结构。

三大特性：

- 稀疏连接：每个输出只连接局部输入，减少参数。
- 参数共享：同一卷积核在整幅图像上滑动，检测同类局部模式。
- 等变表示：输入平移时特征图也相应平移。

基本结构：

- 卷积层：提取局部特征。
- 激活层：引入非线性。
- 池化层：降采样、减少计算、增强局部平移不变性。
- 全连接层：整合特征用于分类或回归。

卷积参数：

- 步长 `stride` 控制卷积核移动距离。
- 填充 `padding` 控制输出尺寸。
- 多通道输入需要三维卷积核。
- 多个卷积核产生多个输出通道。

输出尺寸公式：

`out = floor((in + 2p - k) / s) + 1`

其中 `in` 为输入尺寸，`p` 为填充，`k` 为卷积核尺寸，`s` 为步长。

## 第 5 章 图像压缩编码

### 1. 压缩的必要性

图像数据量大，直接存储和传输成本高。压缩利用图像中的冗余减少比特数。

常见冗余：

- 编码冗余：灰度符号的编码长度不合理。
- 像素间冗余：相邻像素高度相关。
- 心理视觉冗余：人眼对某些误差不敏感。

### 2. 压缩分类

- 无损压缩：解码后图像与原图完全一致，如哈夫曼编码、算术编码、某些预测编码。
- 有损压缩：允许一定失真以获得更高压缩比，如 JPEG。

### 3. 熵编码

熵表示平均自信息量，是无失真编码平均码长的理论下界。若平均码长 `R` 接近熵 `H`，编码效率高。

重要关系：

- `R >> H`：编码效率低。
- `R = H`：理想最佳编码。
- `R < H`：无法无失真表示，必然丢失信息。

### 4. 哈夫曼编码

哈夫曼编码按符号概率构造变长前缀码。概率大的符号码字短，概率小的符号码字长。

步骤：

1. 将符号按概率排序。
2. 合并两个最小概率节点。
3. 重复直到形成二叉树。
4. 从根到叶分配 0/1 得到码字。

### 5. 算术编码

算术编码把整个消息序列表示为 `[0, 1)` 上的一个子区间。随着符号输入，区间不断缩小，最终取区间内任一二进制小数作为编码。

特点：

- 从整体序列出发，通常比逐符号码更接近熵极限。
- 对概率分布建模要求较高。

### 6. 图像质量评价

客观评价：

- MSE：均方误差，越小越好。
- PSNR：峰值信噪比，越大通常越好。
- SSIM：结构相似性，更关注亮度、对比度和结构。

主观评价：

- MOS：平均意见分。
- DMOS：差分平均意见分。

注意：MSE/PSNR 实现简单，但不一定符合人眼主观感受；深度学习质量评价可学习与 MOS/DMOS 更一致的评分函数。

## 第 6 章 图像增强和复原

### 1. 增强与复原区别

- 图像增强：改善视觉效果或突出感兴趣信息，不一定追求真实退化模型。
- 图像复原：根据退化模型估计原始图像，更强调物理/数学建模。

### 2. 灰度修正与同态增晰

灰度修正常用于改善对比度和动态范围，如线性拉伸、直方图均衡化等。同态增晰把图像看作照明分量与反射分量的乘积，通过对数变换转为加性模型，在频域中压低低频照明、增强高频反射。

### 3. 噪声分析与去噪

常见噪声包括高斯噪声、椒盐噪声、乘性噪声等。去噪方法包括均值滤波、中值滤波、频域滤波、DnCNN 等深度去噪网络。

DnCNN 重点：

- 用 CNN 学习噪声残差或干净图像。
- 可通过人工退化生成训练对，如加高斯噪声、JPEG 压缩、降采样。
- 优点是端到端、表达能力强；缺点是依赖训练数据和退化一致性。

### 4. 锐化

图像模糊通常意味着高频成分被衰减。锐化通过增强高频或边缘信息使细节更清晰。

常见方法：

- 梯度法：一阶导数，突出灰度突变。
- Roberts、Sobel、Prewitt 等算子：不同差分模板近似梯度。
- 拉普拉斯算子：二阶导数，对细节和噪声敏感。
- 高通滤波：频域增强高频成分。

注意：锐化前通常应先去噪，否则噪声也会被增强。

### 5. 几何校正与伪彩色

几何校正用于修正图像几何畸变，通常需要控制点、坐标变换和灰度插值。伪彩色增强把灰度映射为颜色，有助于观察细节和分类差异。

## 第 7 章 图像分割

### 1. 分割定义

图像分割是按一定规则把图像划分为若干有意义区域或对象的过程。当感兴趣对象被分离出来时，分割可停止。

两类基本思想：

- 基于不连续性：检测边缘、线、点等灰度突变。
- 基于相似性：把灰度、颜色、纹理等相似像素归为同一区域。

### 2. 边缘检测

边缘是图像灰度发生显著变化的位置。常用一阶梯度或二阶导数检测。

常见算子：

- Roberts：对角差分，简单但抗噪较弱。
- Prewitt/Sobel：利用邻域差分，Sobel 对中心行/列加权。
- Laplacian：二阶导数，检测零交叉，对噪声敏感。
- Marr/LoG：先高斯平滑再用拉普拉斯检测零交叉，更符合视觉特性。

Marr 边缘检测基本思想：

1. 用高斯函数平滑图像。
2. 计算拉普拉斯或 LoG。
3. 找零交叉点作为边缘。

### 3. Hough 变换

Hough 变换把图像空间中的点映射到参数空间，用累加器寻找直线、圆等参数化形状。适合检测被噪声或断裂影响的规则曲线。

### 4. 阈值分割

阈值分割根据灰度阈值把像素分为目标和背景。

方法：

- 全局阈值：整幅图用同一阈值。
- 局部/自适应阈值：不同区域用不同阈值。
- 直方图谷底法：适合目标和背景直方图呈双峰。
- Otsu：最大类间方差法，自动选择阈值。

### 5. 区域增长与分裂合并

区域增长从种子点出发，把满足相似性准则的邻域像素加入区域。关键在于种子点选择和相似性准则。

分裂合并法先把图像划分为区域，对不均匀区域继续分裂，对相邻且满足同质性准则的区域合并，常用四叉树思想。

### 6. 数学形态学与 UNet

数学形态学基于结构元素处理形状。腐蚀会缩小目标，膨胀会扩大目标；开运算先腐蚀后膨胀，可去除小目标；闭运算先膨胀后腐蚀，可填补小孔洞。

UNet 是典型语义分割网络：

- 编码器提取上下文语义。
- 解码器恢复空间分辨率。
- 跳跃连接融合浅层细节与深层语义。

## 第 8 章 图像描述与分类

### 1. 图像描述

图像描述用一组数量或符号描述物体的性质或关系。分割后通常需要描述区域的几何、拓扑、边界、纹理等特征，再用于识别。

### 2. 二值图像几何特征

常见特征：

- 面积：区域像素数。
- 周长：边界长度。
- 质心：区域坐标平均位置。
- 外接矩形：目标的范围。
- 圆形度/紧致度：描述形状接近圆的程度。
- 矩特征：描述区域分布，可构造平移、旋转、尺度不变特征。

### 3. 拓扑特征与邻接

邻接关系常包括 4 邻接、8 邻接等。拓扑特征关注连通分量、孔洞数、欧拉数等，不随连续形变改变。

### 4. 形状描述

区域描述关注区域内部像素分布；边界描述关注目标轮廓。常见边界描述包括链码、边界长度、曲率、傅里叶描述子等。

### 5. 纹理描述

纹理反映局部灰度或颜色的重复模式。常见方法：

- 统计法：灰度共生矩阵、直方图统计。
- 结构法：纹理基元及其排列。
- 频谱法：傅里叶或小波特征。
- 深度特征：CNN 自动学习纹理与语义特征。

### 6. 图像识别与统计模式识别

图像识别通常包括特征提取、分类器设计、训练与测试。统计模式识别根据样本特征和概率/决策函数完成类别判断。

常见分类方法：

- 最小距离分类。
- 贝叶斯分类。
- 线性分类器。
- Softmax 多分类。
- 神经网络分类。

## 第 9 章 目标检测与实例分割

### 1. 目标检测定义

目标检测是在图像或视频中定位并分类一个或多个物体。输出通常包括边界框坐标和类别。

检测子任务：

- 分类：判断框内物体类别。
- 回归：预测边界框位置，如 `(x1, y1, x2, y2)` 或中心点宽高。

### 2. 目标检测与其他任务区别

- 图像分类：只判断整图类别。
- 目标检测：判断类别并给出边界框。
- 语义分割：给每个像素分配类别，不区分同类实例。
- 实例分割：给每个实例输出独立像素掩码。

### 3. R-CNN 系列

R-CNN：

- 用选择性搜索生成候选区域。
- 对每个候选区域提取 CNN 特征。
- 分类和边界框回归分开处理。
- 缺点是速度慢、训练流程复杂。

Fast R-CNN：

- 先对整图提取特征图。
- 将候选区域映射到特征图。
- 用 RoI Pooling 得到固定尺寸特征。
- 分类和回归可端到端训练。

Faster R-CNN：

- 引入 RPN 生成候选框。
- 共享卷积特征，提高速度。

### 4. IoU、NMS 与 mAP

IoU 是预测框和真实框交集面积除以并集面积，是检测评价基础。

混淆矩阵：

- TP：分类正确且 IoU 达到阈值。
- FP：错误检测或 IoU 不达标。
- FN：漏检的真实目标。
- TN：目标检测中通常不显式统计。

NMS 非极大值抑制用于去除重叠冗余框：按置信度排序，保留最高分框，删除与其 IoU 高于阈值的其他框，重复直到结束。

mAP：

- AP 是单类在固定 IoU 阈值下 P-R 曲线下的面积。
- mAP 是各类别 AP 的平均。
- `mAP@0.5` 使用 IoU=0.5。
- `mAP@0.5:0.95` 对多个 IoU 阈值求平均，更严格。

### 5. 实例分割与 Mask R-CNN

实例分割在检测基础上加入像素级掩码预测。Mask R-CNN 在 Faster R-CNN 上增加一个掩码分支，用全卷积网络预测每个 RoI 的实例掩码。

核心改进：

- 并行输出类别、边界框和掩码。
- 用 RoI Align 替代 RoI Pooling，避免量化误差。
- 双线性插值保留空间对齐信息，更适合像素级分割。

局限：

- 计算和显存开销较大。
- 小目标表现可能较差。
- 需要像素级实例掩码标注，数据成本高。
- 复杂背景和边缘模糊会影响掩码质量。

实例分割评价：

- Mask IoU：预测掩码与真实掩码的交并比。
- AP50：掩码 IoU 阈值 0.5，较宽松。
- AP75：掩码 IoU 阈值 0.75，更关注边界精度。
- Mask AP：多个掩码 IoU 阈值下 AP 的平均。

## 高频公式与易混点

- 卷积输出尺寸：`floor((in + 2p - k) / s) + 1`。
- 压缩比：原始数据量 / 压缩后数据量。
- 编码效率：`eta = H / R`。
- MSE：像素误差平方的平均值。
- PSNR：`10 log10(MAX^2 / MSE)`。
- IoU：交集面积 / 并集面积。
- Softmax：把多分类输出归一化为概率，所有类别概率和为 1。

易混点：

- 增强不一定恢复真实图像，复原更依赖退化模型。
- 低通滤波平滑去噪但模糊边缘，高通滤波锐化但增强噪声。
- 语义分割不区分同类实例，实例分割区分每个对象。
- RoI Pooling 有量化误差，RoI Align 用插值减少对齐误差。
- `mAP@0.5:0.95` 基于边界框 IoU，Mask AP 基于掩码 IoU。
- DCT 是固定余弦基，K-L/PCA 的基由数据协方差决定。
