## 8.1 计算图(A)中图像的面积、周长、分散度及伸长度

图(A)的二值图像矩阵（7行×11列，`*`=1，`.`=0）：

```
..*******..
..*******..
***********
***********
..*******..
..*******..
..*******..
```

这是一个十字形（宽矩形 + 窄矩形交叉）的对称连通区域。

**面积**：面积等于目标像素的总个数。

$$A = 7 \times 3 + 7 \times 2 + 11 \times 2 = \text{逐行统计:}\ 7+7+11+11+7+7+7 = 57$$

$$\boxed{A = 57}$$

**周长**：采用4-邻域边界边计数法，对每个目标像素检查上下左右四个邻居，若邻居为背景或越界则计一条边界边。

上下两段（Row0-1, Row4-6）各有7个像素宽，中间段（Row2-3）有11个像素宽，因此在宽度突变处会产生额外的竖直边界边。逐像素统计得：

$$\boxed{P = 36}$$

**分散度**（紧凑度）：

$$C = \frac{P^2}{A} = \frac{36^2}{57} = \frac{1296}{57} \approx 22.74$$

$$\boxed{C \approx 22.74}$$

圆形的分散度为 $4\pi \approx 12.57$，图(A)的值大于此，说明形状偏离圆形，但相比更不规则的形状已较为紧凑。

**伸长度**：利用二阶中心矩的特征值之比来衡量。由对称性可知质心在图像中心 $(\bar{x}=5, \bar{y} \approx 2.93)$，且 $\mu_{11} = 0$（左右对称）。

归一化二阶中心矩：

$$\mu_{20} = \frac{1}{A}\sum(y_i - \bar{y})^2 \approx 3.50, \quad \mu_{02} = \frac{1}{A}\sum(x_i - \bar{x})^2 \approx 6.32, \quad \mu_{11} = 0$$

由于 $\mu_{11}=0$，特征值直接等于 $\mu_{20}$ 和 $\mu_{02}$：

$$\lambda_1 = \mu_{02} \approx 6.32, \quad \lambda_2 = \mu_{20} \approx 3.50$$

$$\boxed{E = \frac{\lambda_{\max}}{\lambda_{\min}} = \frac{6.32}{3.50} \approx 1.80}$$

伸长度约1.80，说明图(A)在水平方向上比垂直方向略为伸展。

## 8.3 图(C)所示16×16图像，右上角坐标为(0,0)，利用低阶矩求其质心坐标

图(C)是一个16×16的二值图像：

```
Row 0:  ................   (0个)
Row 1:  .......**.......   (2个, col7-8)
Row 2:  ...*.******.....   (7个, col3,5-10)
Row 3:  ....********....   (8个, col4-11)
Row 4:  ***************.   (15个, col0-14)
Row 5:  ..************..   (12个, col2-13)
Row 6:  ..***********...   (11个, col2-12)
Row 7:  ...*********....   (9个, col3-11)
Row 8~15: 全0
```

坐标系：右上角为原点 $(0,0)$。对于第 $i$ 行第 $j$ 列（从0开始）的像素，坐标为 $x = 15 - j$（向左为正），$y = i$（向下为正）。

**零阶矩（面积）**：

$$m_{00} = \sum\sum f(x,y) = 2+7+8+15+12+11+9 = 64$$

**一阶矩**：

$$m_{10} = \sum\sum x \cdot f(x,y) = 502$$

$$m_{01} = \sum\sum y \cdot f(x,y) = 289$$

**质心坐标**：

$$\bar{x} = \frac{m_{10}}{m_{00}} = \frac{502}{64} \approx 7.84$$

$$\bar{y} = \frac{m_{01}}{m_{00}} = \frac{289}{64} \approx 4.52$$

$$\boxed{\text{质心坐标} \approx (7.84,\ 4.52)}$$

## 8.5 对二类特征、二类模式情形，举出无法用线性决策函数进行分类的例子

**经典例子：异或（XOR）问题。**

设有二维特征空间 $(x_1, x_2)$，两个类别 $\omega_1$ 和 $\omega_2$ 的样本分布如下：

$$\omega_1: \{(0,0),\ (1,1)\}$$
$$\omega_2: \{(0,1),\ (1,0)\}$$

在二维平面上，$\omega_1$ 类的两个样本分布在主对角线上，$\omega_2$ 类的两个样本分布在副对角线上。无论怎样画一条直线 $w_1 x_1 + w_2 x_2 + w_0 = 0$，都无法将 $\omega_1$ 和 $\omega_2$ 完全分开。

这是因为异或问题是"线性不可分"的：两类样本在特征空间中交叉排列，不存在一个线性超平面能够将它们正确划分。这也是早期感知器模型的著名局限性。要解决此类问题，需要采用非线性决策函数（如二次判别函数）或引入核函数将数据映射到高维空间使其线性可分。

## 8.6 对二分类问题，若先验概率是等概分布，讨论最小错误概率的Bayes决策的四种形式的相互关系，说明采用对数似然比有何优点

设有两个类别 $\omega_1$ 和 $\omega_2$，先验概率相等：$P(\omega_1) = P(\omega_2) = \frac{1}{2}$。

### Bayes决策的四种等价形式

**形式一：基于后验概率的决策**

$$\text{若 } P(\omega_1|x) > P(\omega_2|x) \Rightarrow x \in \omega_1, \quad \text{否则 } x \in \omega_2$$

由 Bayes 公式 $P(\omega_i|x) = \frac{p(x|\omega_i)P(\omega_i)}{p(x)}$，当先验等概时 $P(\omega_1) = P(\omega_2)$，分母 $p(x)$ 相同，上式化简为比较类条件概率密度。

**形式二：基于类条件概率密度（似然函数）的决策**

$$\text{若 } p(x|\omega_1) > p(x|\omega_2) \Rightarrow x \in \omega_1, \quad \text{否则 } x \in \omega_2$$

由于先验概率相等，后验概率之比只取决于似然函数之比，因此直接比较两类的类条件概率密度即可。

**形式三：基于似然比的决策**

定义似然比 $l(x) = \frac{p(x|\omega_1)}{p(x|\omega_2)}$，则判决规则为：

$$l(x) = \frac{p(x|\omega_1)}{p(x|\omega_2)} \underset{\omega_2}{\overset{\omega_1}{\gtrless}} \frac{P(\omega_2)}{P(\omega_1)}$$

当先验等概时，阈值 $\frac{P(\omega_2)}{P(\omega_1)} = 1$，即：

$$\text{若 } l(x) > 1 \Rightarrow x \in \omega_1, \quad \text{否则 } x \in \omega_2$$

**形式四：基于对数似然比的决策**

对似然比取对数，定义对数似然比 $h(x) = \ln l(x) = \ln p(x|\omega_1) - \ln p(x|\omega_2)$，则：

$$h(x) = \ln p(x|\omega_1) - \ln p(x|\omega_2) \underset{\omega_2}{\overset{\omega_1}{\gtrless}} \ln \frac{P(\omega_2)}{P(\omega_1)}$$

当先验等概时，阈值 $\ln 1 = 0$，即：

$$\text{若 } h(x) > 0 \Rightarrow x \in \omega_1, \quad \text{否则 } x \in \omega_2$$

### 四种形式的相互关系

这四种形式在数学上完全等价，当先验概率相等时尤其简洁。形式一是最基本的 Bayes 决策表述；由于先验等概，形式一简化为形式二（直接比较似然函数）；形式二中的不等式两边同除以 $p(x|\omega_2)$ 就得到形式三（似然比判决）；形式三两边取对数就得到形式四（对数似然比判决）。它们是同一判决规则的不同数学表达。

### 对数似然比的优点

1. **将乘除运算转化为加减运算**：当类条件概率密度为多维正态分布等指数族分布时，对数运算可以消去指数函数，使判别函数变为特征向量的多项式函数，极大简化了计算。

2. **数值计算更稳定**：直接计算似然函数 $p(x|\omega_i)$ 时，多维高斯分布的概率密度值可能极小（接近下溢）或差别极大，取对数后数值范围更合理，避免了浮点溢出或下溢问题。

3. **便于分析判别函数的几何意义**：取对数后，若类条件概率密度为正态分布，则对数似然比 $h(x)$ 为 $x$ 的二次函数（一般情况）或线性函数（等协方差矩阵情况），可以直观地分析决策面的几何形状（超平面或二次超曲面）。

4. **单调变换不改变判决结果**：对数函数是严格单调递增函数，因此 $l(x)>1$ 与 $h(x)>0$ 完全等价，取对数不会改变分类结果，却能带来上述计算和分析上的便利。
